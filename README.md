
# AVozer

<p>AVozer is an Android app that was created as a part of the Android development course of HIT (Holon Institute of Technology). The app is designed to help people find babysitters, cleaners, and dog walkers. With AVozer, users can post help requests, specifying the type of help they need, the date and time they need it, and the city they're located in. Other users can then browse these requests and offer to help.</p>


## Installation

To install AVozer, follow these steps:


1. Download the APK file from the [releases page](https://github.com/VladimirPoplavsky/AVozer/releases/tag/APK).
2. Open the APK file on your Android device.
3. Follow the installation prompts.
## Usage
To use AVozer, follow these steps:
1. Open the app on your Android device.
2. Register new account or log in if you already have one.
3. Press "I Want To Help" to browse the list of requests to see if there are any you can help with.
4. If you find a request you can help with, click on it to view more details.
4. If you need to create a new request, click the "I Need Help" button and fill out the form.
